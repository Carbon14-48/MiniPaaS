# API Reference

All services expose OpenAPI docs at `/docs` when running.

## Contracts

- `service-auth-contract-v1.md` - shared service-to-service auth standard

## Service Ports

| Service | Port | Docs URL |
|---------|------|----------|
| API Gateway | 8000 | http://localhost:8000/docs |
| Auth Service | 8001 | http://localhost:8001/docs |
| App Management | 8002 | http://localhost:8002/docs |
| Build Service | 8003 | http://localhost:8003/docs |
| Deployment Service | 8004 | http://localhost:8004/docs |
| Monitoring Service | 8005 | http://localhost:8005/docs |
| Security Scanner | 8006 | http://localhost:8006/docs |
